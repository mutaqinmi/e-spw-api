# E - SPW Web
Website monitoring aktivitas pembelajaran PKK (Projek Kreatif dan Kewirausahaan) siswa untuk guru.

## Deskripsi
E - SPW Web adalah website monitoring aktivitas dalam pembelajaran PKK (Projek Kreatif dan Kewirausahaan) siswa yang dikhususkan bagi guru. Website ini membantu guru dalam melihat laporan hasil penjualan maupun data keuangan serta mengatur kelompok dalam kegiatan PKK.

## Fitur Unggulan
- All in One, pengelolaan kelompok serta laporan dikemas dalam satu website yang fleksibel.
- Simple and Fast, tampilan yang disederhanakan membuat pengelolaan semakin cepat.
- See All Reports, semua laporan di generate secara terstruktur dan mudah diakses.
- Manage, atur semuanya dalam genggaman anda.
